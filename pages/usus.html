USUS (Usage)

§ I

Type System

Veritas passes C types through directly after stripping the English article. Multi-word types and pointers are supported.

VeritasC Equivalent
a stringchar* (Managed)
an intint
a floatfloat
a double complexdouble complex
a double pointerdouble*
nothingvoid

§ II

Variable Creation

Create 'name' as a string.
Create 'x' as a double with value 3.14.
Create 'c' as a double complex with value 1.0 plus 2.0j.

§ III

Array Creation & List Rules

Lists must follow strict formatting rules:

  • 1 item: X
  • 2 items: X and Y
  • 3+ items: X, Y, and Z
Create 'data' as a double array of size 3 with values:
1.0, 2.0, and 3.0.

§ IV

Conditionals

If 'name' is equal to "Jose":
    Call 'printf' with "Hello, Jose!
", stored to nothing.
End if 'name' is equal to "Jose".
