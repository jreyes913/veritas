USUS (Usage)

§ I

Type System

Veritas passes C types through directly after stripping the English article. Multi-word types and pointers are supported.

VeritasC Equivalent
a stringchar* (Managed)
an intint
a floatfloat
a double complexdouble complex
a vectorNumeric-only sequence
a matrixMathematical data structure
nothingvoid

§ II

Variable Creation

Create 'name' as a string.
Create 'x' as a double with value 3.14.
Create 'c' as a double complex with value 1.0 plus 2.0j.
/* Indexed creation */
Create 'val' as an element of 'data' at index 0.

§ III

Data Structures & List Rules

Lists follow strict grammatical rules based on item count:

  • 1 item: X
  • 2 items: X and Y (No commas)
  • 3+ items: X, Y, and Z (Oxford comma required)
/* Vector (Numeric) */
Create 'v' as a double vector of size 3 with values:
1.0, 2.0, and 3.0.

/* Array (Non-numeric) */
Create 'labels' as a string array of size 2 with values:
"Initial" and "Result".

/* Matrix */
Create 'dataset' as a matrix with columns: "ID" and "Value".

§ IV

Arithmetic & Exponents

Standard operators are supported, including the specialized power operator.

Replace 'y' with 'x' raised to the 2.
Replace 'v_sum' with the quantity 'v1' plus 'v2'.

§ V

Data Persistence (I/O)

Load and save matrices from external files. Supports CSV and raw Binary formats.

/* Load from CSV */
Load 'data' from "input.csv" as matrix.

/* Load from Binary (requires dimensions) */
Load 'raw' from "data.bin" as bin matrix with size 100 by 3.

/* Save to CSV */
Save 'data' to "output.csv" as matrix.
