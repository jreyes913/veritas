EXEMPLA (Examples)

§ I

Finance: Compound Interest

This is the program 'finance'.

Create 'principal' as a double with value 1000.0.
Create 'rate' as a double with value 0.05.
Create 'years' as an int with value 5.

For every iteration of 'y' from 1 through 'years':
    Replace 'principal' with 'principal' multiplied by the quantity 1.0 plus 'rate'.
    Call 'printf' with "Year %d: %.2f
" and 'y' and 'principal', stored to nothing.
End iteration of 'y' from 1 through 'years'.

End of the program 'finance'.

§ II

Physics: Free Fall

This is the program 'physics'.

Create 'g' as a double with value 9.81.
Create 't' as a double with value 2.5.
Create 'height' as a double.

/* h = 0.5 * g * t^2 using exponents */
Create 't_sq' as a double with value 't' raised to the 2.
Replace 'height' with 0.5 multiplied by 'g' multiplied by 't_sq'.

Call 'printf' with "Distance: %.2f meters
" and 'height', stored to nothing.

End of the program 'physics'.

§ III

Math: Vector Operations

This is the program 'vectors'.

Create 'v1' as a double vector of size 3 with values: 1.0, 2.0, and 3.0.
Create 'v2' as a double vector of size 3 with values: 10.0, 20.0, and 30.0.
Create 'res' as a double vector of size 3.

Replace 'res' with the quantity 'v1' plus 'v2'.

End of the program 'vectors'.
