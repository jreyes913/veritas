INITIUM (Getting Started)

§ I

Structure of a Program

Every Veritas program must be enclosed in matching declaration statements. The program name must be consistent at the start and end.

This is the program 'hello_world'.

/* Your instructions go here */

End of the program 'hello_world'.

§ II

The Hello World Example

Veritas handles basic I/O through the standard C library. Note that the common stdio.h is included automatically by the intelligent prelude.

This is the program 'hello'.

Call 'printf' with "Hello, Veritas!
", stored to nothing.

End of the program 'hello'.

§ III

Sentence Termination

Consistency is the core of truth. Every instruction must end with a period (.). You may split a single instruction across multiple lines for readability; the compiler will treat it as a single sentence until it encounters the period.

Create 'x'
as an int
with value 42.
